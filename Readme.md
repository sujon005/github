# Student Skill Marketplace

A Java-based console application that connects skilled students with job opportunities.
The system automatically matches students to jobs based on required skills.

## Features
- Student registration with skills
- Job posting by clients
- Skill-based auto matching
- Clean OOP design

## Technologies
- Java
- OOP
- Collections (ArrayList)

## Future Enhancements
- Login system (Student / Client)
- Match percentage logic
- File storage
- Web version using Spring Boot
